Date Visualizer

This widgit pulls 1 date field from a redcap instance and visualizes it as a line graph, bucketed to every day, and a bar chart, bucketed to every month. Please note that by entering your API token this site could potentially perform a malicious action, you should review the JS code (its short) on this site before using or adapt/reuse the code for your own deploy.
